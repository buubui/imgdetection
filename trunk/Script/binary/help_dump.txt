dump_rhog
=========
	Compute RHOGDense features over images

Description: 
Compute RHOG fetures using image gradient strength over image window. The whole algorithm is:
1) divide image window into uniformly spaced dense points with point spacing given by 'descstride'
2) for each point, compute 3-dimensional position-orientation gradient histogram
3) finally create a big feature vector containing descriptor vector from each point in the window


Usage: dump_rhog.exe [options]  infile  outfile 

Options common to config file and command line:
  --dumphard  (Default: 0)                               infile is hard 
                                                         examples file
  -W [ --window ]  (Default: [ 64,128 ]) (Min: [ 3,3 ])  window width,height
  --winstride  (Default: [ 8,8 ])                        window stride along 
                                                         x-y
  --scaleratio  (Default: 1.05) (Min: 1)                 scale ratio
  --endscale                                             scale-space pyramid 
                                                         end scale
                                                           (Default: till image
                                                         size >= window size)
  --startscale  (Default: 1) (Min: 1)                    start scale
  --topleft                                              top-left 
                                                         X,Y-coordinate of 
                                                         descriptor window
                                                           (Default window is 
                                                         at image center )
  --fullsize                                             full window 
                                                         width,height
                                                           Use iff top-left is 
                                                         valid, not used for 
                                                         hard examples
  --cachesize  (Default: 128) (Min: 4)                   descriptor cache size 
                                                         (im MB) to use
                                                           NOTE: small cache 
                                                         size can drastically 
                                                         reduce speed
  -v [ --verbose ]  (Default: 0) (Min: 0, Max: 9)        verbose level
  --testlocs                                             filename where to 
                                                         write all test or 
                                                         feature dump locations
                                                           (Format: imagename X
                                                         Y Width Height)
  -x [ --imageext ]                                      image extension (valid
                                                         only if 'infile' = dir
                                                         or list)
  -f [ --dumpdetail ]  (Default: 0) (Min: 0, Max: 3)     dump extra information
                                                         about the feature in 
                                                         the file
  -s [ --samples ]  (Default: 1) (Min: 0)                number of random 
                                                         sampling windows over 
                                                         each image OR  if hard
                                                         is true, number of 
                                                         samples to skip
  -r [ --random ]  (Default: 0)                          initialize random 
                                                         generator with current
                                                         time
  --hardscore  (Default: 0)                              use only points with 
                                                         score greater than 
                                                         hardscore
  --poscases                                             number of positive 
                                                         cases
  --negcases                                             number of negative 
                                                         cases
  --memorylimit                                          upper memory limit on 
                                                         training examples
  --jitterwinstep  (Default: 0) (Min: 0)                 jitter window steps
  -l [ --label ]  (Default: O) (Min: N, Max: P)          hard file labels 
                                                         (Default is ignore 
                                                         label)
  -p [ --pyramid ]  (Default: 0)                         compute descriptors 
                                                         over scale-space 
                                                         pyramid
  --proc  (Default: RGB_Sqrt)                            Image preprocessor and
                                                         gradient computation 
                                                         method to use
  --norm  (Default: L2Hys)                               Descriptor normalizing
                                                         method to use
  -C [ --cellsize ]  (Default: [ 8,8 ]) (Min: [ 1,1 ])   cell size, position 
                                                         invariance tolerance 
                                                         in each cell
  -N [ --numcell ]  (Default: [ 2,2 ]) (Min: [ 1,1 ])    number of cells in 
                                                         each block
  -G [ --descstride ]  (Default: [ 8,8 ]) (Min: [ 1,1 ]) descriptor stride in 
                                                         window
  -B [ --orientbin ]  (Default: 9) (Min: 1)              number of orientation 
                                                         bin
  --wtscale  (Default: 2) (Min: 0)                       magnitude weighting 
                                                         scale
  -O [ --fullcirc ]                                      take orientations in 
                                                         range (0-360)
  -S [ --gscale ]  (Default: 0) (Min: 0)                 gradient computation 
                                                         scale
  --epsilon  (Default: 1) (Min: 0)                       epsilon to add while 
                                                         normalizing
  --maxvalue  (Default: 0.2) (Min: 0, Max: 1)            chop feature vector 
                                                         values > maxvalue 
                                                         after normalizing
                                                           Used iff normalizing
                                                         method == NormL2Hys

Command line options only:
  -h [ --help ]         show help screen
  --version             print version number and contact details

Required Arguments:
  --infile              input image file/dir/list
  --outfile             out file

Usage Issues:
options 'stride', 'startscale', 'scaleratio' and 'endscale'  are valid iff 'pyramid' is true'
  'infile'         can be an image file, a directory, or a list file.
  'imageext'       empty string implies all listed files, otherwise images with given extension are selected
   'dumpdetail'    dump window detail in the output file(currently does not work for hard examples or with pyramid)
                   0 - Write only descriptor
                   1 - Write descriptor top-left, extent, scale
                   2 - Write filename information also
Supported options in image preprocessor are:
       RGB
       Lab
       RGB_Sqrt
       Lab_Sqrt
       RGB_Log
Supported options in descriptor normalizer are:
       NoNorm
       L1
       L2
       L1Sqrt
       L2Hys
